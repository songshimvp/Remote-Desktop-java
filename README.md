# Remote-Desktop-java
Application remote an other computer using socker programming RMI in java

# 项目介绍：

Java Remote Desktop是一个Java 的远程桌面软件，支持例如文件传输、数据压缩、颜色转换、键盘鼠标事件转换等等很多特性。
